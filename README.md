# DOCUMENTOS PROYECTO PYHTON

Resolucion ejercicios Python

📌 Descripción

Este repositorio contiene una colección de ejercicios desarrollados en Python, enfocados en fortalecer habilidades de programación, lógica y buenas prácticas de desarrollo.

Los ejercicios abarcan desde conceptos básicos hasta aplicaciones prácticas, incluyendo estructuras de datos, control de flujo, funciones, manipulación de datos

🎯 Objetivos

-Reforzar fundamentos de programación en Python

-Aplicar lógica y resolución de problemas

-Implementar buenas prácticas de código

-Documentar y estructurar proyectos correctamente

🛠️ Tecnologías Utilizadas

-Python
-Jupiter Notebook en VSCode

📚 Contenido de los Ejercicios

-Manipulación de variables y estructuras básicas

-Operadores aritméticos, lógicos y de comparación

-Uso de funciones y condicionales

-Trabajo con listas, tuplas y diccionarios

-Manejo de errores

-Estructuras condicionales (if, elif, else)

-Métodos de listas (append, remove, sort, etc.)

-Iteración con bucles for

-Validación de datos ingresados por el usuario
